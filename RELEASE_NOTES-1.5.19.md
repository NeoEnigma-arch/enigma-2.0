# Enigma 2.0 – Release Notes 1.5.19

## Deutsch

### Behoben / verbessert (Voice-Modem)
- **WAV laden** funktioniert zuverlässig: eigener **Voice-Fenster**-Workflow (Firefox schließt sonst das Popup beim Dateidialog).
- Nach erfolgreicher Entschlüsselung öffnet sich ein **Klartext-Fenster**.
- **Aufnahme-Anzeige:** rotes Banner „AUFNAHME AKTIV“, Sekundenzähler und **Mikrofon-Pegelbalken**.
- Decoder robuster bei leichten Tempo-/Sample-Abweichungen.

### Kurzversion für AMO
```
Fix: Voice-Modem mit eigenem Fenster (WAV/Aufnahme), sichtbarer Aufnahme-Status und Klartext-Ergebnis-Fenster.
```

## English

### Fixed / improved (voice modem)
- **Load WAV** is reliable via a dedicated **voice window** (Firefox closes the popup on file dialogs).
- Successful decrypt opens a **plaintext result window**.
- **Recording UI:** red “RECORDING LIVE” banner, timer, and mic level meter.
- Decoder more robust to slight speed/sample-rate drift.

### Short version for AMO
```
Fix: dedicated voice-modem window (WAV/record), visible recording status, and plaintext result window.
```
