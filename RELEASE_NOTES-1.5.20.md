# Enigma 2.0 – Release Notes 1.5.20

## Deutsch
- Modem-Audio **deutlich kürzer**: Standard jetzt **160 baud** (statt 40) → etwa **~4× schneller**
- Geschwindigkeitswahl im Audio-Fenster: **Fast / Normal / Robust**
- Decoder erkennt weiterhin alte langsame WAVs (40 baud)

## English
- Modem audio much shorter: default **160 baud** (~4× faster than 40)
- Speed selector: **Fast / Normal / Robust**
- Decoder still reads older slow WAVs
