# Enigma 2.0 – Release Notes 1.5.16

## Deutsch

### Neu / Design
- **Popup-Hintergrund schwarz** statt Dunkelblau.
- **Animierter Matrix-Regen** (Katakana & Symbole) als vollflächiger Hintergrund im Toolbar-Popup.
- UI bleibt durch leichte Abdunklung über dem Regen gut lesbar.

### Kurzversion für AMO
```
Design: Popup mit schwarzem Hintergrund und animiertem Matrix-Regen (volle Fläche, grüne Zeichen).
```

---

## English

### New / Design
- **Popup background is pure black** instead of slate blue.
- **Animated Matrix rain** (katakana & symbols) as a full-area toolbar popup background.
- Light overlay keeps controls and text readable over the rain.

### Short version for AMO
```
Design: popup uses a pure black background with full-area animated Matrix rain.
```
